# Platform Specifications and Best Practices

> Format requirements and best practices per channel. Load at boot.

## Platform Summary

| Platform | Primary Audience | Content Focus | Posting Cadence |
|----------|-----------------|--------------|----------------|
| Blog | Organic search visitors | SEO-optimized long-form | 3-4 posts/week |
| LinkedIn | B2B professionals | Thought leadership, value posts | 1-2 posts/day |
| X (Twitter) | Industry peers, tech audience | Hot takes, threads, news | 3-5 tweets/day |
| Instagram | Visual-first audience | Carousels, stories, reels | 1 post/day + stories |
| Email/Newsletter | Subscribers | Curated insights, exclusives | 1/week |

## Blog

### Format Specs
- Min word count: 800 words (SEO threshold)
- Ideal word count: 1200-2000 words (blog), 3000-5000 words (guides)
- Featured image: 1200 x 630px (WebP/PNG)
- Meta title: under 60 characters
- Meta description: under 160 characters
- URL slug: short, keyword-rich, no stop words

### Publishing Checklist
- [ ] Title and meta description optimized
- [ ] Featured image with alt text
- [ ] Internal links (2+ minimum)
- [ ] External links (1+ to authoritative source)
- [ ] Categories and tags assigned
- [ ] Schema markup added
- [ ] Social sharing image (OG) configured

### SEO Requirements
- Primary keyword in: title, H1, first 100 words, meta description
- Secondary keywords in: H2/H3, naturally throughout
- One H1 per page
- Subheadings every 300 words for scannability

## LinkedIn

### Content Formats
| Format | Dimensions | Best For |
|--------|-----------|----------|
| Text post | N/A | Thought leadership, stories, opinions |
| Single image | 1200 x 1200px | Data visualization, quotes |
| Carousel (PDF) | 1080 x 1350px | Step-by-step, frameworks |
| Video | 1920 x 1080px (landscape) | Tutorials, announcements |
| Article | N/A | Long-form (lower reach than posts) |

### Best Practices
- Text posts outperform image posts for reach (currently)
- First line is the hook -- make it count
- Line breaks between every 1-2 sentences (readability on mobile)
- 1200-1500 characters optimal for text posts
- 3-5 hashtags, mix of broad and niche
- Post Tue-Thu 8-10am or 12-1pm (audience's timezone)
- Reply to every comment in the first hour

## X (Twitter)

### Content Formats
| Format | Specs | Best For |
|--------|-------|----------|
| Single tweet | 280 chars | Quick insights, hot takes |
| Thread | 5-12 tweets | Deep dives, tutorials |
| Image tweet | 1200 x 675px | Data, quotes, infographics |
| Quote tweet | 280 chars + source | Commentary, amplification |

### Best Practices
- Threads outperform single tweets for engagement (5-12 tweets ideal)
- First tweet is everything -- most people won't click "Show more"
- Number tweets in threads: "1/8", "2/8" etc.
- Last tweet: summary + CTA + link
- 1-2 hashtags max (or none -- hashtags feel spammy on X now)
- Post Mon-Fri 8-10am, 12pm, 5-6pm

## Instagram

### Content Formats
| Format | Dimensions | Best For |
|--------|-----------|----------|
| Post (square) | 1080 x 1080px | Product shots, quotes |
| Carousel | 1080 x 1350px | Tutorials, lists, stories |
| Reel | 1080 x 1920px (9:16) | Short educational content |
| Story | 1080 x 1920px | Behind-scenes, polls, daily updates |

### Best Practices
- Carousels get highest engagement (saves + shares)
- First slide is the hook (treat like a thumbnail)
- 7-10 slides optimal for carousels
- Hashtags in first comment, not caption (10-20 hashtags)
- Post Tue-Fri 11am-1pm or 7-9pm
- Stories daily for algorithm favor
- Saves and shares matter more than likes

## Email / Newsletter

### Format Specs
- Subject line: under 50 characters, curiosity-driven
- Preheader: under 90 characters, complements subject
- Width: 600px max (email client safe)
- Images: include alt text (many clients block images by default)
- CTA: one primary CTA per email, above the fold
- Unsubscribe: always included, easy to find

### Best Practices
- Send Tuesday or Thursday morning (highest open rates)
- Personalize subject line when possible
- Plain text outperforms HTML for engagement in many B2B segments
- Keep to 500-800 words
- One topic per newsletter (don't split attention)
- P.S. line gets disproportionate attention (use it for secondary CTA)

## Cross-Platform Rules

1. **Never identical cross-post** -- each platform gets native content
2. **Adapt the core message** -- same insight, different packaging
3. **Respect platform culture** -- what works on LinkedIn doesn't work on X
4. **Track separately** -- measure each platform's metrics independently
5. **Prioritize by audience** -- if most of your audience is on LinkedIn, LinkedIn gets the best content first
