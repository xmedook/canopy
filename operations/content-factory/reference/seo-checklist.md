# SEO Optimization Checklist

> Complete checklist for optimizing content before publication.
> Load when optimizing content for search.

## Pre-Writing SEO

- [ ] Primary keyword identified with search volume and difficulty
- [ ] Search intent classified (informational, commercial, transactional)
- [ ] Top 3 ranking pages analyzed (format, length, angle, gaps)
- [ ] Secondary keywords identified (2-5 related terms)
- [ ] Content format matches search intent
- [ ] Existing content checked (update existing vs create new)

## On-Page Optimization

### Title Tag
- [ ] Primary keyword included (front-loaded if natural)
- [ ] Under 60 characters
- [ ] Compelling to click (not just keyword-stuffed)
- [ ] Unique across the site (no duplicate titles)

### Meta Description
- [ ] Under 160 characters
- [ ] Includes primary keyword
- [ ] Contains a CTA or value proposition
- [ ] Reads like a compelling pitch, not a summary

### URL
- [ ] Short and descriptive
- [ ] Includes primary keyword
- [ ] No stop words (the, and, of, in)
- [ ] Lowercase, hyphens between words

### Headings
- [ ] One H1 per page (matches title, includes primary keyword)
- [ ] H2s for main sections (include secondary keywords where natural)
- [ ] H3s for sub-sections as needed
- [ ] Heading hierarchy is logical (no skipping H2 to H4)

### Content Body
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword density 1-2% (natural, never forced)
- [ ] Secondary keywords distributed naturally throughout
- [ ] Content length matches or exceeds top-ranking competitors
- [ ] Reads naturally -- keyword integration doesn't feel forced

### Internal Links
- [ ] 2+ internal links to related content
- [ ] Anchor text is descriptive (includes relevant keywords)
- [ ] Links to pillar page from cluster content
- [ ] No broken internal links

### External Links
- [ ] 1+ links to authoritative external sources
- [ ] External links set to open in new tab
- [ ] Links to current, relevant resources (not outdated)

### Images
- [ ] All images have descriptive alt text
- [ ] Primary keyword in at least one image alt text (if relevant)
- [ ] Images compressed for web (WebP preferred)
- [ ] Image filenames are descriptive (not IMG_0001.jpg)
- [ ] Featured image is 1200 x 630px

## Technical SEO

- [ ] Page loads in under 3 seconds
- [ ] Mobile-responsive (test on actual mobile device or emulator)
- [ ] Schema markup added (Article, FAQ, HowTo, etc.)
- [ ] Canonical URL set (prevents duplicate content)
- [ ] No noindex tag (unless intentional)
- [ ] XML sitemap includes this page
- [ ] Robots.txt allows crawling

## Content Quality Signals

- [ ] Content is original (not substantially similar to existing pages on our site)
- [ ] Provides unique value (original data, expert insight, or novel angle)
- [ ] E-E-A-T signals present (Experience, Expertise, Authoritativeness, Trustworthiness)
- [ ] Author byline with credentials
- [ ] Published date and last-updated date visible
- [ ] Sources cited for claims and statistics

## Post-Publish SEO

### Within 24 Hours
- [ ] Verify page is indexed (Google Search Console)
- [ ] Verify schema markup renders correctly (Rich Results Test)
- [ ] Verify no crawl errors
- [ ] Share on social media (initial traffic signal)

### Within 7 Days
- [ ] Check initial ranking position for target keyword
- [ ] Verify internal links from other pages point to this content
- [ ] Monitor Core Web Vitals for the page

### Within 30 Days
- [ ] Check ranking movement
- [ ] Review search queries driving impressions (Search Console)
- [ ] If not ranking: diagnose (intent mismatch? authority gap? technical issue?)
- [ ] If ranking but low CTR: optimize title tag and meta description

### Within 90 Days
- [ ] Full performance review
- [ ] If ranking top 10: optimize for featured snippet
- [ ] If not ranking: consider content update or new angle
- [ ] Update with any new information or data points

## Common SEO Mistakes to Avoid

| Mistake | Why It Hurts | Fix |
|---------|-------------|-----|
| Keyword stuffing | Google penalizes unnatural keyword use | Write naturally, use synonyms |
| Thin content | Short pages with little value rank poorly | Add depth, examples, data |
| Duplicate content | Splits ranking signals between pages | Canonical tags, consolidate |
| Missing alt text | Missed image search opportunity + accessibility fail | Add descriptive alt text |
| Slow page speed | Google uses speed as ranking factor | Compress images, minimize JS |
| No internal links | Google can't discover or understand page relationships | Link related content |
| Ignoring search intent | Wrong format for the query = won't rank | Match intent before writing |
