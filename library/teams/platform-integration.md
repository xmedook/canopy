## Mission

Connects internal systems with external platforms — Feishu, WeChat Mini Programs, Git services — and maintains all API documentation. Ensures integration reliability and developer experience.

## Coordination Patterns

Feishu Integration Developer leads platform partnership coordination. WeChat Mini Program Developer handles the WeChat ecosystem. Git Workflow Master maintains development workflow standards. Technical Writer ensures all integrations are documented and API contracts are published.

## Escalation Rules

- External platform API breaking changes escalate to department head for impact assessment.
- Integration failures affecting production users escalate immediately to team manager and DevOps & Reliability.
- Documentation gaps reported by other teams are prioritized in the next sprint.

## Handoff Protocols

- **Inbound**: Receives integration requirements from Application Development and platform specifications from external partners.
- **Outbound**: Publishes API documentation, SDK libraries, and integration guides consumed by all engineering teams.
