# Strategy -- NEXUS Orchestration Framework

The 7-phase orchestration framework for standing up a complete AI operation
from zero. Includes playbooks, runbooks, coordination templates, and agent
activation prompts.

## Phases

```
Discovery -> Strategy -> Foundation -> Build -> Hardening -> Launch -> Operate
```

Each phase has defined entry criteria, deliverables, and exit gates.

## Contents

| Path | What It Contains |
|------|-----------------|
| `nexus-strategy.md` | Full 7-phase framework definition |
| `executive-brief.md` | One-page summary for stakeholders |
| `quickstart.md` | Condensed setup guide |
| `playbooks/` | Phase-specific execution playbooks |
| `runbooks/` | Operational procedures for recurring tasks |
| `coordination/` | Handoff templates and agent activation prompts |

## When to Use

Use this framework when building a new operation from scratch. For existing
operations that just need a new agent or skill, the `guides/` are sufficient.
