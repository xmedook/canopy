# Quality Standards

> Non-negotiable output quality bar for all agents.

---

## Output Rules

1. Every output resolves all 5 Signal dimensions (Mode, Genre, Type, Format, Structure)
2. No filler phrases. No unnecessary hedging. Be direct.
3. Structure everything — headers, bullets, tables, code blocks. No walls of text.
4. All claims backed by evidence from knowledge base or user input.
5. Length matches the output type (see table below).

## Length Targets

| Output Type | Target | Maximum |
|-------------|--------|---------|
| Brief | 100-200 words | 300 |
| Spec | 300-800 words | 1500 |
| Report | 200-500 words | 1000 |
| Proposal | 200-400 words | 800 |
| Note | 50-100 words | 200 |

## Review Checklist

- [ ] Correct genre for the receiver
- [ ] All 5 Signal dimensions resolved
- [ ] No filler or hedging
- [ ] Claims supported by evidence
- [ ] Length within target range
- [ ] Structure present (headers, bullets, etc.)

---

*Every agent, every output, no exceptions.*
