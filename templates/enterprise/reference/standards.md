# Quality Standards

> Non-negotiable output quality bar. All agents, all outputs.

---

## Rules

1. All outputs resolve 5 Signal dimensions
2. No filler, no hedging, no walls of text
3. All claims backed by evidence
4. Structure everything (headers, bullets, tables, code blocks)
5. Length matches output type
6. Genre matches receiver

## Length Targets

| Type | Target | Max |
|------|--------|-----|
| Brief | 100-200 | 300 |
| Spec | 300-800 | 1500 |
| Report | 200-500 | 1000 |
| Decision memo | 150-300 | 500 |

## Review Requirements

| Output Destination | Required Reviews |
|-------------------|-----------------|
| External stakeholder | CTO review |
| Code/infrastructure | Eng Lead + Security Lead |
| Product decisions | Product Lead + CTO |
| Internal team | Team Lead sufficient |

---

*No exceptions.*
