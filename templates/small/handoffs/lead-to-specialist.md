# Handoff: Lead → Specialist

> Template for delegating work from the Lead to the Specialist agent.

## Handoff Format

```markdown
## Task Handoff

**From**: Lead
**To**: Specialist
**Priority**: [high | normal | low]
**Deadline**: [date/time or "none"]

### What I Need
[Clear, specific description of the deliverable]

### Constraints
- [Constraint 1 — what's off the table]
- [Constraint 2 — hard requirements]

### Context
[Relevant background the Specialist needs. Keep it minimal — link to reference
files rather than copying content.]

### Expected Output
- **Genre**: [spec | analysis | code | report]
- **Length**: [approximate]
- **Format**: [what it should look like]

### When You're Done
Return the deliverable to me for review. Flag any risks or open questions.
```

## Rules

1. Every handoff MUST include "What I Need" and "Constraints" sections.
2. Context should reference files, not duplicate them.
3. Expected Output must specify genre — the Specialist needs to know what form.
4. If the Specialist needs clarification, they ask ONCE, then execute.
